# PagePulse — Why Is This Page Slow?

PagePulse is a Chrome extension that analyzes webpage performance and explains why a page is slow using Computer Network concepts.

## 🚀 Features
- Real-time performance analysis
- Detects issues like slow DNS, high TTFB, heavy pages
- Provides fixes and explanations
- Visual network timeline

## 🛠️ Technologies Used
- JavaScript
- Chrome Extension APIs
- Performance API
- HTML, CSS

## 📚 Concepts Covered
- DNS Resolution
- TCP Handshake
- TLS Handshake
- HTTP Protocols
- Latency & Bandwidth

## 📦 How to Use
1. Download the project
2. Open Chrome → Extensions → Enable Developer Mode
3. Click "Load unpacked"
4. Select this folder
